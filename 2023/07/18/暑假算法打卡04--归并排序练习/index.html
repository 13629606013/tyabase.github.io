# 归并排序练习

Acwing 样题： Acwing.788 逆序对的数量

我的想法：

  1. 定义了全局变量 q 数组用于存储输入的数据， tmp 数组用于存储归并排序时的临时数据， n 表示输入数据的个数， ans 是一个长整型变量用于记录逆序对数量。
  2. merge_sort 函数实现了归并排序算法。在每次调用时，根据左边界 l 和右边界 r 进行判断。如果 l 大于等于 r ，意味着当前待排序的区间只有一个元素或者为空，不需要再进行拆分和排序，所以直接返回。
  3. 然后计算中间位置 mid = l + r >> 1 ，将数组 q 拆分为两部分：左半部分是从左边界 l 到中间位置 mid ，右半部分是从中间位置 mid+1 到右边界 r 。
  4. 继续递归地对左半部分和右半部分分别调用 merge_sort 函数，进行排序。
  5. 创建临时数组 tmp ，初始化三个指针 k 、 i 、 j 分别为 0、 l 、 mid+1 。
  6. 通过比较左半部分和右半部分的元素大小，将较小的元素依次放入临时数组 tmp 中，并移动相应的指针。如果当前选择的右半部分的元素 q[j] 小于左半部分的元素 q[i] ，则说明右半部分存在与左半部分元素构成逆序对的情况，因此需要统计逆序对数量并更新 ans 。
  7. 接着处理剩余的元素，将左半部分或右半部分中剩余的元素依次放入临时数组 tmp 中。
  8. 最后，将排好序的临时数组 tmp 中的元素复制回原数组 q 中。
  9. 在 main 函数中，首先读取输入数据个数 n ，然后依次读取数据填充到数组 q 中。
  10. 调用 merge_sort 函数对数组 q 进行归并排序，同时会在排序过程中统计逆序对数量并更新 ans 。
  11. 最后，输出逆序对的数量 ans 。

该代码通过归并排序算法的思想，在排序的过程中统计了逆序对的个数，并将结果输出。逆序对是指在一个数组中，如果两个元素的索引位置与元素值的大小顺序不一致，则称它们构成一个逆序对。因此，这段代码实现了通过归并排序来高效地计算逆序对数量的功能。

<font color=red> 解释说明：</font>

在归并排序算法中，每当将一个右半部分元素 q[j] 放入临时数组 tmp 时，说明存在一对逆序对。因为在归并排序的过程中，左半部分和右半部分都是有序的，所以当 q[j] 比左半部分的某个元素 q[i] 小的时候，意味着 q[j] 同时小于 q[i] 及其后面的所有元素，即构成逆序对。

而在当前情况下，左半部分的索引范围是从 l 到 mid ，所以逆序对的数量就是右半部分元素 q[j] 相对于左半部分剩余元素数量的个数，即 (mid - i + 1) 。

因此，在代码中 ans += mid - i + 1 用于统计逆序对的数量，每次碰到右半部分小于左半部分的元素时，将逆序对的数量加上 (mid - i + 1) 。最终的 ans 存储了整个数组中的逆序对数量。

#include<iostream>
using namespace std;
int q[100010], tmp[100010], n;
long long ans;

void merge_sort(int l, int r) {
    if (l >= r) return;

    int mid = l + r >> 1;

    merge_sort(l, mid);
    merge_sort(mid + 1, r); 

    int k = 0, i = l, j = mid + 1;

    while (i <= mid && j <= r) {
        if (q[i] <= q[j])
            tmp[k++] = q[i++];
        else {
            tmp[k++] = q[j++];
            ans += mid - i + 1;
        }
    }

    while (i <= mid)
        tmp[k++] = q[i++];
    while (j <= r)
        tmp[k++] = q[j++];

    for (i = l, j = 0; i <= r; i++, j++)
        q[i] = tmp[j];
}


int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> q[i];

    merge_sort(0, n);

    cout << ans;
}
